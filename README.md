# Url
The url class allows you to interact with the URLs.

```php
$url_base = Url::getBase();
$url_current = Url::getCurrent();
$uri_string = Url::getUriString();
$uri_segments = Url::getUriSegments();
$uri_segment = Url::getUriSegment(0);
```
